# Clinical Trial Eligibility Checker — AI-Powered Screening System

> An AI-assisted clinical trial screening system that combines deterministic rule logic with LLM reasoning to evaluate single patients and entire cohorts at scale, producing structured, explainable eligibility decisions instantly.

🔗 **[Live Demo](https://adrianjrosario8-clinical-trial-eligibility-checker-app-xvqghk.streamlit.app/)** - Deployed and fully functional on Streamlit Cloud

---

## Problem Statement

Clinical trial screening is manual, time-consuming, and prone to inconsistency. Teams must evaluate inclusion criteria, exclusion criteria, and clinical edge cases across hundreds of patients, creating bottlenecks in recruitment and trial efficiency that delay time-to-market and increase costs.

---

## Solution

A hybrid decision engine that separates deterministic clinical logic from LLM reasoning, ensuring auditability and consistency with a design principle aligned with regulatory requirements in clinical settings.

```
Patient Data → Rule-Based Validation → LLM Reasoning (Groq) → Structured JSON Output → Streamlit UI
```

- **Rule-based layer** enforces clinical constraints deterministically (inclusion/exclusion thresholds, comorbidity checks)
- **LLM layer** (Groq LLaMA 3) generates human-readable clinical reasoning, only after rules are enforced
- **Result:** Every decision is traceable, explainable, and consistent

## Key Features

**Batch Cohort Processing**
Upload a CSV of multiple patients and screen an entire cohort simultaneously with results downloadable as Excel for clinical documentation

**Single Patient Evaluation**
Real-time eligibility check for individual patient records with instant explainable output

**Hybrid Decision Engine**
Deterministic rule enforcement + LLM reasoning in a clean two-layer architecture and no black-box decisions

**Explainable Outputs**
Every decision includes an inclusion/exclusion validation flag and a one-line clinical reason

**Robust Data Handling**
Column normalization, missing value handling, and flexible CSV input formats for real-world messy data
